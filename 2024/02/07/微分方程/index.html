微分方程

发表于 2024-02-07 阅读次数： 本文字数： 3.7k 阅读时长 ≈ 3 分钟

一阶线性微分

可分离变量类

形如$\frac{\mathrm{d}y}{\mathrm{d}x}=f(x)g(y)：略$

直接分部积分法

一阶微分方程拥有显式的分部积分关系：$u\frac{\mathrm{d}v}{\mathrm{d}x}+v\frac{\mathrm{d}u}{\mathrm{d}x}=\frac{\mathrm{d}}{\mathrm{d}x}(uv)=f(x)$；两边同时积分$\int u\frac{\mathrm{d}v}{\mathrm{d}x}+\int v\frac{\mathrm{d}u}{\mathrm{d}x}=uv=\int f(x)$；即根据$uv=\int f(x)$解出y的值（u,v是y的函数）
ex:$x^3\frac{\mathrm{d}y}{\mathrm{d}x}+3x^2y=sinx$
解：$\begin{aligned}
&u=x^3\text{, }v=y\text{,则: }uv=x^3y=\int sinxdx=-cosx+c\mathrm{~.} \
&\text{得:}\quad y=-\frac{cosx}{x^3}+\frac c{x^3}.
\end{aligned}$

「凑」分部积分法

形如$\frac{\mathrm{d}y}{\mathrm{d}x}+f(x)y=g(x)$的1阶微分方程的分部积分关系不是显式的；
令$h(x)\frac{\mathrm{d}y}{\mathrm{d}x}+h(x)f(x)y=g(x)h(x)$；存在（1）$u’(x)=h’(x)=h(x)f(x)$关系;
根据（1），$e^{\int f(x)dx}=h(x)$
⇒$ e^{\int f(x)dx} \frac{\mathrm{d} y}{\mathrm{d} x} + e^{\int f(x)dx} f(x)y=g(x) e^{\int f(x)dx} $
根据分部积分$uv=\int g(x)h(x)$
⇒$e^{\int f(x)dx}y=\int g(x)e^{\int f(x)dx}$
⇒$ y=\frac{\int g(x) e^{\int f(x)dx}dx}{e^{\int f(x)dx}} $

二阶线性微分

微分算子是线性算子：$ \forall k\in R,\frac{\mathrm{d} }{\mathrm{d} x} (\sum_{i=1}^{n} k_i\cdot f_{i}(x))=\sum_{i=1}^{n} k_i\frac{\mathrm{d} f_{i}(x)}{\mathrm{d} x} $

常系数齐次微分$a\frac{\mathrm{d} ^2y}{\mathrm{d} x^2} +b\frac{\mathrm{d} y}{\mathrm{d} x} +cy=0；a≠0$

（1）解向量组的秩为2？？？？？
（2）特解$y_1=A$$e^{kx}$(猜的)；另一解$y_2$必须要与$y_1$线性无关才能构成通解：$\frac{y_1}{y_2}必不能为常数$

1）令$y=e^{kx}\quad,\quad\frac{\mathrm{d}y}{\mathrm{d}x}=ke^{kx}\quad,\quad\frac{\mathrm{d}^2y}{\mathrm{d}x^2}=k^2e^{kx}$；待定系数
⇒$ ak^2e^{kx}+bke^{kx}+ce^{kx}=0 $
⇒$e^{kx}(ak^2+bk+c)=0$
解出特征根(系数)：
$\Delta > 0$：含有两个实根（$\alpha$，$\beta$）：得到$y_1=e^{\alpha x}$和$y_2=e^{\beta x}$：$ {\color{Red} y \color{Red} = \color{Red} A \color{Red} e^{\color{Red}\alpha \color{Red} x} \color{Red} + \color{Red} B \color{Red} e^{\color{Red}\beta \color{Red} x} } $（需$\frac{e^{\alpha x}}{e^{\beta x}} \ne c$）；$A,B\in R$$\Delta =0$：含重根$y_{1}=e^{\alpha x}$；（$\frac{e^{\alpha x}}{e^{\alpha x}} =1$显然不行）
设$\frac{y_2}{y_1}= \varphi(x)$：寻找另一线性无关解${y_2}= \varphi(x) {y_1}= \varphi(x)e^{kx}$：
待定系数：$e^{kx}(a {\varphi}’’(x)+2ak {\varphi}’(x)+ak^2 {\varphi}(x)+b {\varphi}’(x)+bk {\varphi}(x) +c{\varphi}(x))=0$
⇒解特征方程$(ak^2+bk+c) {\varphi}(x)+(2ak+b) {\varphi}’(x)+a {\varphi}’’(x)=0$
⇒$(2ak+b) {\varphi}’(x)+a {\varphi}’’(x)=0$；k是根知道可消
⇒${\varphi}’’(x)=0$
⇒${\varphi}’(x)=\int{\varphi}’’(x)dx=c_1$
⇒${\varphi}(x)=\int{\varphi}’(x)dx=c_1x+c_2$
⇒$y_2= (c_1x+c_2)y_1= (c_1x+c_2)e^{\alpha x}$
随便代个数给$c_1,c_2$⇒$y_2=xe^{\alpha x}$
⇒${y}=Ay_1+By_2=Ae^{\alpha x}+Bxe^{\alpha x} {=} {(} { A} { +} {B} {x} {)} {e}^{\alpha{x}}$

$\Delta < 0$:有两个不同的共轭虚根系数$p+qi$，$p-qi$：
$y_1=e^{(p+qi)x}= e^{px}\cdot e^{iqx}$=$e^{px}(cosqx+isinqx)$$y_2=e^{px}\cdot e^{-iqx}=e^{px}(cosqx-isinqx)$
到这一步其实就可以了，但还可以构造两个形式更好的线性无关解：
$y_3=\frac{1}{2}(y_1+y_2)=e^{px}cosqx$$y_4=\frac{1}{2i}(y_1-y_2)=e^{px}sinqx$$y=Ay_3+By_4=Ae^{px}cosqx+Be^{px}sinqx=(Acosqx+Bsinqx)e^{px}$

常系数非齐次微分$a\frac{\mathrm{d} ^2y}{\mathrm{d} x^2} +b\frac{\mathrm{d} y}{\mathrm{d} x} +cy=f(x)；a≠0$

尝试从齐次方程解推广到非齐：
1）假设有特解$y_1$使之成立：$a\frac{\mathrm{d} ^2}{\mathrm{d} x^2}y_{1} +b\frac{\mathrm{d} }{\mathrm{d} x} y_{1}+cy_{1}=f(x)$
令$a\frac{\mathrm{d} ^2y}{\mathrm{d} x^2} +b\frac{\mathrm{d} y}{\mathrm{d} x} +cy$-$a\frac{\mathrm{d} ^2}{\mathrm{d} x^2}y_{1} +b\frac{\mathrm{d} }{\mathrm{d} x} y_{1}+cy_{1}=f(x)$$-f(x)$构造齐次方程
⇒$a\frac{\mathrm{d} ^2}{\mathrm{d} x^2}(y-y_{1}) +b\frac{\mathrm{d} }{\mathrm{d} x} (y-y_{1}) +c(y-y_{1}) =0$；可见齐次解$y_{2}=y-y_{1}$与通解的关系
y的解为齐次解$y_2$+特解$y_1$
2）对于特解：前辈们一个个试出来了。。。。

知道特解的形式后，可以用待定系数法，将特解回代到原方程中，得到未知的系数；

还有一种情况要注意：特解与齐次解的某一项线性相关（必与齐次解线性相关），那么给特解乘个x
