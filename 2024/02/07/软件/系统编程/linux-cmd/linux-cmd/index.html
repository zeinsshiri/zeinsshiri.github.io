直接键入程序名shell会去PATH找这个程序

注意：

如果没有指定查找目录，则为当前目录。

$ find . -name “ *.txt “ //其中.代表当前路径。

$ find -name “ *.txt “ //没加路径，默认是当前路径下查找。

find 还有一些高级的用法，如查找最近几天(几个小时)之内(之前)有变动的文件

$ find /home/book -mtime -2 //查找/home 目录下两天内有变动的文件。
