bmp

发表于 2024-02-08 分类于 数字图像处理 阅读次数： 本文字数： 8.3k 阅读时长 ≈ 8 分钟

采用位映射存储格式，图像深度可选：1bit、4bit、8bit及24bit；不采用其他任何压缩，因此，BMP文件所占用的空间很大。
BMP文件存储数据时，图像的扫描方式是按从左到右、从下到上的顺序。由于BMP文件格式是Windows环境中交换与图有关的数据的一种标准

BMP文件格式详解（bmp类）

BMP文件由四个部分构成：

（1）BMP文件头(bmp file header)：固定占用14字节,用于标识文件格式和一些基本信息。

  • 文件类型（2字节）：标识文件为BMP格式，通常为”BM”。
  • 文件大小（4字节）：指定整个BMP文件的大小，以字节为单位。
  • 保留字段（4字节）：保留字段，通常设置为0。
  • 数据偏移量（4字节）：指定图像像素数据相对于文件起始位置的偏移量。

（2）位图信息头(bitmap information)：提供图像数据的尺寸、位平面数、压缩方式、颜色索引等信息

  • 信息头大小（4字节）：指定信息头的大小。
  • 图像宽度（4字节）：图像的宽度，以像素为单位。
  • 图像高度（4字节）：图像的高度，以像素为单位。
  • 颜色平面数（2字节）：总为1。
  • 颜色位深度（2字节）：指定每个像素的位深度，即用于表示颜色的比特数。
  • 压缩类型（4字节）：指定图像数据的压缩类型，常见的有无压缩、RLE压缩等。
  • 图像大小（4字节）：指定图像数据的大小，以字节为单位。若使用无压缩存储，通常为0。
  • 水平分辨率（4字节）：图像的水平分辨率，以像素每米为单位。
  • 垂直分辨率（4字节）：图像的垂直分辨率，以像素每米为单位。
  • 使用的颜色数（4字节）：图像实际使用的调色板中的颜色数目。若为0，则表示使用了所有颜色。
  • 重要颜色数（4字节）：对图像产生重要影响的颜色数目。若为0，则表示所有调色板项都重要。

（3）~~~~调色板~~~~(color palette)：可选，如使用索引来表示图像，调色板就是索引与其对应的颜色的映射表，在使用 256位彩色、16位彩色等情况下用到

（4）位图数据(bitmap data)：存放图像的像素数据；存放顺序从左到右、从下到上的顺序存放。
每一行像素数据需要保证4字节对齐，不对齐的需要补充空字节，这个是BMP文件格式本身要求的。对于RGBA 32位色彩格式来说没有任何问题，但是对于Gray或者RGB格式的数据 ，有时一行像素需要补充一些空字节。
