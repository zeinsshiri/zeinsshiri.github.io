傅里叶分析

发表于 2024-02-08 分类于 math 阅读次数： 本文字数： 8.1k 阅读时长 ≈ 7 分钟

三角函数的正交性：

三角函数系${sin(nx+\frac{k\pi}{2});n,k∈Z}$=${1,\mathrm{cosx,sinx,cos2x,sin2x,\ldots,cosnx,sinnx,~\ldots}}$中任取两元素（任意两个三角函数）:
$f(x)={f(x);x∈[-\pi,\pi]},g(x)={g(x);x∈[-\pi,\pi]}$
两函数内积（即向量各分量的线性组合）$ \lim_{\lambda \to +\infty}\sum_{x=-\pi}^{\pi-\frac{1}{\lambda } } f(x)g(x)\frac{1}{\lambda } $为0;
其所有情况：
$ \begin{aligned}&\int_{-\pi}^{\pi}\sin mx\cdot\cos nxdx:=:0\\&\int_{-\pi}^{\pi}\cos mx\cdot\cos nxdx:=:0\\&\int_{-\pi}^{\pi}\sin mx\cdot\sin nxdx:=:0\end{aligned} $；（n≠m）
if n=m，存在$\int_{-\pi}^{\pi}\cos mx\cdot\cos nxdx:≠:0$⇒故取n≠m情况；
综上：任意两个不同的三角函数在$[-\pi,\pi]$上正交;

周期函数在一个周期上的积分与起点无关：

$要证\int_a^{a+T}f(x)\mathrm{d}x=\int_0^Tf(x)\mathrm{d}x$
⇒$\int_a^{a+T}f(x)\mathrm{~d}x$=$ \int_a^0f(x)\mathrm{d}x+\int_0^Tf(x)\mathrm{d}x+\int_T^{a+T}f(x)\mathrm{d}x
$；要证$\int_a^0f(x)\mathrm{d}+\int_T^{a+T}f(x)\mathrm{d}x=0$
⇒令x=u+T;$\int_a^0f(x)\mathrm{dx}+\int_0^{a}f(u+T)\mathrm{d}u=$$\int_a^0f(x)\mathrm{dx}-\int_a^{0}f(u)\mathrm{d}u=0$

傅里叶级数

将时域上周期为T的函数 分解为 整数倍角频率$n\omega=n\frac{2\pi}{T}$三角波（谐波）的叠加（频域函数，甚至复频域）；

强版本：周期为$2\pi$ 的周期函数展开为傅里叶级数:
$f(x)=\sum_{n=0}^{+\infty}a_{n}\cos nx+\sum_{n=0}^{+\infty}b_{n}\sin nx=$$\frac{a’{0}}{2}+\sum{n=1}^{\infty}(a_{n}\cos nx+b_{n}\sin nx)$;($a’0=2a_0$)
利用三角函数正交性消项获得系数：
1）$\int{-\pi}^{\pi} f(x)dx$=$\int_{-\pi}^{\pi}\frac{a’{0}}{2}dx+\sum{n=1}^{+\infty}\int_{-\pi}^{\pi}a_{n}\cos nxdx+\sum_{n=1}^{+\infty}\int_{-\pi}^{\pi}b_{n}\sin nxdx$
⇒$a’0/2=\frac{1}{2\pi}$$\int{-\pi}^{\pi} f(x)dx$

2)$\int_{-\pi}^{\pi} f(x)cosmxdx$=$\int_{-\pi}^{\pi}\frac{a’{0}}{2}cosmxdx+\sum{n=1}^{+\infty}\int_{-\pi}^{\pi}a_{n}\cos nx\cdot cosmxdx+\sum_{n=1}^{+\infty}\int_{-\pi}^{\pi}b_{n}\sin nxcosmxdx$
⇒$\int_{-\pi}^{\pi} f(x)cosmxdx=\int_{-\pi}^{\pi}\sum_{n=1}^{+\infty}a_{n}\cos nx\cdot cosmxdx$;(n≠0的项的积分都为0)
⇒$\int_{-\pi}^{\pi} f(x)cosnxdx=\int_{-\pi}^{\pi}a_{n}\cos nx\cdot cosnxdx$
⇒$\int_{-\pi}^{\pi} f(x)cosnxdx=\int_{-\pi}^{\pi}a_{n}\frac{1+cos2x}{2}dx$
⇒$a_n=\frac{1}{\pi}\int_{-\pi}^{\pi} f(x)cosnxdx$

3）$b_n=\frac{1}{\pi}\int_{-\pi}^{\pi} f(x)sinnxdx$

放宽条件的弱版本：周期为T ∈R的周期函数展开为傅里叶级数:
周期T的g(t)由周期$2\pi$的 f(t)函数拉伸了$\frac{T}{2\pi}$倍；同相位自变量变换因子$\omega$=$\frac{2\pi}{T}$：g(t)$:=$ f($\frac{2\pi}{T}$t)
$g(t)=f($$\frac{2\pi}{T}$$t$)=$\sum_{n=0}^{+\infty}a_{n}\cos n\frac{2\pi}{T}t+\sum_{n=0}^{+\infty}b_{n}\sin n\frac{2\pi}{T}t$=$\frac{a’{0}}{2}+\sum{n=1}^{\infty}(a_{n}\cos n\frac{2\pi}{T}t+b_{n}\sin n\frac{2\pi}{T}t)$
利用三角函数正交性消项获得系数：
1)$ a’0/2=\frac{1}{2T}\int{-T/2}^{T/2} g(t)dt $=$ \frac{1}{2T}\int_{0}^{T} g(t)dt $
2)$a_n=\frac{1}{T}\int_{-T/2}^{T/2} g(t)cosn\frac{2\pi}{T}tdt=\frac{1}{T}\int_{0}^{T} g(t)cosn\frac{2\pi}{T}tdt$
3)$b_n=\frac{1}{T}\int_{-T/2}^{T/2} g(t)sinn\frac{2\pi}{T}tdt=\frac{1}{T}\int_{0}^{T} g(t)sinn\frac{2\pi}{T}tdt$

证明：参考苏云答案以及信号与系统

傅里叶级数的复数形式：
$g(t)=$$\frac{a’{0}}{2}+\sum{n=1}^{\infty}(a_{n}\cos n\frac{2\pi}{T}t+b_{n}\sin n\frac{2\pi}{T}t)$=$\frac{a’{0}}{2}$+$\sum{n=1}^{+\infty}(a_{n}\frac{e^{jn\frac{2\pi}{T}t}+e^{-jn\frac{2\pi}{T}t}}{2}+b_{n}\frac{e^{jn\frac{2\pi}{T}t}-e^{-jn\frac{2\pi}{T}t}}{2j})$
=$\frac{a’{0}}{2}$+$\sum{n=1}^{+\infty}(a_{n}\frac{e^{jn\frac{2\pi}{T}t}+e^{-jn\frac{2\pi}{T}t}}{2}-jb_{n}\frac{e^{jn\frac{2\pi}{T}t}-e^{-jn\frac{2\pi}{T}t}}{2})$=$\frac{a’{0}}{2}$+$
\sum{n=1}^{+\infty}\frac{a_{n}-jb_{n}}{2}e^{jn\frac{2\pi}{T}t}+
\sum_{n=1}^{+\infty}\frac{a_{n}+jb_{n}}{2}e^{-jn\frac{2\pi}{T}t} $
令n=-n：
=$\sum_{n=0}^{0}\frac{a’{0}}{2}e^{jn\frac{2\pi}{T}t}+$$ $$\sum{n=1}^{+\infty}\frac{a_{n}-jb_{n}}{2}e^{jn\frac{2\pi}{T}t}+ \sum_{n=-1}^{-\infty}\frac{a_{-n}+jb_{-n}}{2}e^{jn\frac{2\pi}{T}t}$
=$\sum_{-\infty}^{+\infty}C_ne^{jn\frac{2\pi}{T}t}$

$C_n=\begin{cases}
\\frac{a’{0}}{2}=\frac{1}{2T}\int{0}^{T} g(t)dt ;n=0\
\\frac{a_{n}-jb_{n}}{2}=\frac{1}{T}\int_{0}^{T} g(t)(cosn\frac{2\pi}{T}t- jsinn\frac{2\pi}{T}t)dt=\frac{1}{T}\int_{0}^{T} g(t)e^{-jn\frac{2\pi}{T}t}dt;n∈Z^+\
\\frac{a_{-n}+jb_{-n}}{2}=\frac{1}{T}\int_{0}^{T} g(t)(cosn\frac{2\pi}{T}t- jsinn\frac{2\pi}{T}t)dt=\frac{1}{T}\int_{0}^{T} g(t)e^{-jn\frac{2\pi}{T}t}dt;n∈Z^-
\end{cases}$
⇒
$C_n=\frac{1}{T}\int_{0}^{T} g(t)e^{-jn\omega t}dt$
g(t)=$\sum_{n=-\infty}^{+\infty}C_ne^{jn\omega t}$=$….+C_0+C_1e^{j1\omega t}+C_2e^{j2\omega t}+…..C_ne^{jn\omega t}+……$=$g(\omega)$

非周期函数(T→$\infty$,$\omega→0$)的傅里叶级数：
$g(t)=\sum_{-\infty}^{+\infty}(\frac{\omega}{2\pi}\int_{-\infty}^{+\infty} g(t)e^{-jn\omega t}dt)e^{jn\omega t}$=$\frac{1}{2\pi}\int _{-\infty}^{+\infty}
[\int _{-\infty}^{+\infty} g(t)e^{-jn\omega t}dt]e^{jn\omega t}

d\omega$=$\frac{1}{2\pi}\int _{-\infty}^{+\infty}
F(\omega)e^{jn\omega t}

d\omega$
$F(\omega)=\int _{-\infty}^{+\infty} g(t)e^{-jn\omega t}dt$;这个式子称为傅里叶变换($s=jn\omega$ 为拉氏变换)
$F^{-1}(\omega)=g(t)=\frac{1}{2\pi}\int _{-\infty}^{+\infty}
F(\omega)e^{jn\omega t}

d\omega$;称为傅里叶逆变换

物理角度

傅里叶算子性质（对T相同输入函数）

$x(t)\overset{FS}{\operatorname*{\longleftrightarrow}}\sum_{n=-\infty}^{+\infty}C_ne^{jn\frac{2\pi}{T}t}=\sum_{n=-\infty}^{+\infty}(\frac{1}{T}\int_{0}^{T} x(t)e^{-jn\frac{2\pi}{T}t}dt)e^{jn\frac{2\pi}{T}t}$;
对于相同T的不同输入函数：$C_n取决于x(t)$，其余部分的形式对任意x(t)都是相同的；简化描述为：$x(t)\overset{FS}{\operatorname*{\longleftrightarrow}}C_n$

线性

input输入函数的线性组合；output输入响应的线性组合：$Ax(t)+By(t)\longleftrightarrow Aa_{k}+Bb_{k}$
简单验证：$C_n=\frac{1}{T}\int_{0}^{T} [Ax(t)+By(t)]e^{-jn\frac{2\pi}{T}t}dt=Aa_{k}+Bb_{k}$

时移

$x(t)\overset{FS}{\operatorname*{\longleftrightarrow}}C_n$；发生时移则：$x(t-t_0)\overset{FS}{\longleftrightarrow}C_ne^{-jn\frac{2\pi}{T}t_0}$
验证：$C’n=\frac{1}{T}\int{0}^{T} x(t-t_0)e^{-jn\frac{2\pi}{T}t}dt$;令$\tau =t-t_0$:$C’n=\frac{1}{T} \int{0}^{T} x(\tau)e^{-jn\frac{2\pi}{T}(\tau +t_0)}d\tau$=$C_ne^{-jn\frac{2\pi}{T}t_0}$

时间反转

$x(-t)\longleftrightarrow C_{-n}$:
$C’n=\frac{1}{T}\int{0}^{T} x(-t)e^{-jn\frac{2\pi}{T}t}dt$:令$\tau =-t$：$C’n=-\frac{1}{T} \int{0}^{-T} x(\tau)e^{-jn\frac{2\pi}{T}(-\tau)}d\tau$=$C’n=\frac{1}{T} \int{0}^{T} x(\tau)e^{j(-n)\frac{2\pi}{T}\tau}d\tau$=$C_{-n}$

时域尺度变换(周期发生变化)

$x(at)\overset{FS}{\operatorname*{\longleftrightarrow}}C_n$;其周期为$\frac{T}{a}$:
验证：$C’n=\frac{1}{T}\int{0}^{T} x(at)e^{-jn\frac{2\pi}{T}t}dt$;令$\tau =at$:$C’n=\frac{1}{aT} \int{0}^{aT} x(\tau)e^{-jn\frac{2\pi}{T}(tau/a)}d\tau$=$\frac{1}{T} \int_{0}^{T} x(\tau)e^{-jn\frac{2a\pi}{T}(tau/a)}d\tau$=$C_n$

相乘

周期为T的输入：
$x(t)y(t)\longleftrightarrow\sum_{l=-\infty}^{\infty}a_lb_{n-l}=a_n*b_n$
验证：$C’n=\frac{1}{T}\int{0}^{T} x(t)y(t)e^{-jn\frac{2\pi}{T}t}dt=\frac{1}{T}\int_{0}^{T} (\sum_{l=-\infty}^{+\infty}a_le^{jl\frac{2\pi}{T}\text{t}})y(t)e^{-jn\frac{2\pi}{T}t}dt=\sum_{l=-\infty}^{+\infty}a_l\frac{1}{T}\int_{0}^{T}y(t)e^{-j(n-l)\frac{2\pi}{T}t}dt=\sum_{l=-\infty}^{+\infty}a_lb_{n-l}$

微分

$\frac{dx(t)}{dt}\overset{FS}{\operatorname*{\longleftrightarrow}}jn\omega_0C_n$
证：$ x(t)=$$ \sum_{n=-\infty}^{+\infty}C_ne^{jn\frac{2\pi}{T}\text{t}} $;两边乘d/dt
⇒$ dx(t)/dt=$$ \sum_{n=-\infty}^{+\infty}jn\omega C_ne^{jn\frac{2\pi}{T}\text{t}} $

共轭

$x^*(t)\longleftrightarrow C_{-n}^*$
证：$ x(t)=$$ \sum_{n=-\infty}^{+\infty}C_ne^{jn\frac{2\pi}{T}\text{t}} $;两边共轭
⇒$ x^*(t)=$$ \sum_{n=-\infty}^{+\infty}C^*ne^{-jn\frac{2\pi}{T}\text{t}} $;令n=-n
⇒$ x^*(t)=$$ \sum{n=-\infty}^{+\infty}C^*_{-n}e^{jn\frac{2\pi}{T}\text{t}} $

若x(t)是实函数:x(t)=$x^*(t)$$C_n=C^*_{-n}$
若x(t)=x(-t)是偶实函数：$C_n=C^*_{-n}=C_{-n}=C^*_{n}$⇒$C_n是实偶函数$
若x(t)=-x(-t)是奇实函数：$C_n=C^*_{-n}=-C_{-n}=-C^*_{n}$⇒$C_n是纯虚奇函数$
